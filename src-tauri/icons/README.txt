Place your icon files here. Generate them from your favicon.png by running:

  npx tauri icon path/to/favicon.png

Required files:
  32x32.png
  128x128.png
  128x128@2x.png
  icon.ico        (Windows)
  icon.icns       (macOS)
  icon.png        (Linux, 512x512 recommended)

The tauri icon command generates all of these automatically.
